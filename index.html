About David

A motivated full-stack JavaScript developer with more than 20 years experience in IT, I have been programming since my teens. I started learning JavaScript in 2009 in order to create a PC Inventory Tracking System in my technology support position at Zones. Streamlining my work through web applications motivated me to study JavaScript full-time in 2016 at Code Fellows in Portland OR. A constant learner, I continue to use my experience in system administration and IT to inform my software development. Dedicated to creating efficient user interfaces and experiences, I focus on test-driven development to create simple elegant code. I thrive on contributing to a positive and co-operative work environment, and would like to bring my talent, skills and ambition to a great company.
